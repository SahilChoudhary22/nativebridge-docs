## Overview

The Activity Dashboard provides a comprehensive view of all your testing activities, enabling you to track performance, identify issues, and optimize your testing workflow. Access detailed logs, session recordings, and analytics for both manual device sessions and automated test runs.

## Accessing the Activity Dashboard

<Steps>
  <Step title="Navigate to Your Profile">
    Click on the profile icon located in the top-right corner of the interface
  </Step>

  <Step title="Select Activity Section">
    Choose **Activity** from the dropdown menu or navigate directly to [nativebridge.io/activity](https://nativebridge.io/activity)
  </Step>
</Steps>

## Data Management

<Tabs>
  <Tab title="Retention Policies">
    - Session logs retained for 30 days
    - Video recordings available for 14 days
    - Test reports archived for 90 days
    - Custom retention available for enterprise plans
  </Tab>

  <Tab title="Export Options">
    - **CSV Export**: Structured data for spreadsheet analysis
    - **JSON Format**: Machine-readable format for integration
    - **PDF Reports**: Professional documentation for stakeholders

## Troubleshooting

<Warning>
  If you experience issues accessing session data or recordings, verify your account permissions and check the retention period for the requested data.
</Warning>

### Common Issues and Solutions

| Issue | Solution |
|-------|----------|
| Missing session recordings | Check if recordings are enabled in project settings |
| Incomplete test logs | Ensure proper test framework integration |
| Slow dashboard loading | Try filtering to a smaller date range |
| Export failures | Verify available storage quota |

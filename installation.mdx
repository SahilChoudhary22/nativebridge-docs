## Choose Your Platform

Select your development platform to see specific installation instructions.

<Tabs>
  <Tab title="React Native">
    ## React Native Installation

    ### Step 1: Install Package

    ```bash
    npm install @nativebridge/react-native
    # or
    yarn add @nativebridge/react-native
    ```

    ### Step 2: iOS Setup

    ```bash
    cd ios && pod install
    ```

    ### Step 3: Android Setup

    Add to your `android/app/build.gradle`:

    ```gradle
    dependencies {
        implementation 'com.nativebridge:android:latest'
    }
    ```

    ### Step 4: Link Native Dependencies

    ```bash
    npx react-native link @nativebridge/react-native
    ```
  </Tab>

  <Tab title="Ionic/Capacitor">
    ## Ionic/Capacitor Installation

    ### Step 1: Install Package

    ```bash
    npm install @nativebridge/capacitor
    ```

    ### Step 2: Sync Native Projects

    ```bash
    npx cap sync
    ```

    ### Step 3: iOS Configuration

    Add to your `ios/App/Podfile`:

    ```ruby
    pod 'NativeBridge', '~> 1.0'
    ```

    ### Step 4: Android Configuration

    The Android dependencies are automatically included.
  </Tab>

  <Tab title="Cordova">
    ## Cordova Installation

    ### Step 1: Install Plugin

    ```bash
    cordova plugin add @nativebridge/cordova
    ```

    ### Step 2: Platform Setup

    ```bash
    cordova platform add ios
    cordova platform add android
    ```

    ### Step 3: Build Project

    ```bash
    cordova build
    ```
  </Tab>

  <Tab title="WebView">
    ## WebView Installation

    ### Step 1: JavaScript Library

    ```html
    <script src="https://cdn.nativebridge.dev/v1/nativebridge.min.js"></script>
    ```

    Or install via npm:

    ```bash
    npm install @nativebridge/webview
    ```

    ### Step 2: iOS WebView Setup

    ```swift
    import WebKit
    import NativeBridge

    class ViewController: UIViewController {
        @IBOutlet weak var webView: WKWebView!

        override func viewDidLoad() {
            super.viewDidLoad()
            NativeBridge.attach(to: webView)
        }
    }
    ```

    ### Step 3: Android WebView Setup

    ```kotlin
    import android.webkit.WebView
    import com.nativebridge.NativeBridge

    class MainActivity : AppCompatActivity() {
        override fun onCreate(savedInstanceState: Bundle?) {
            super.onCreate(savedInstanceState)

            val webView = findViewById<WebView>(R.id.webview)
            NativeBridge.attach(webView)
        }
    }
    ```
  </Tab>
</Tabs>

## Verify Installation

Run this test to confirm NativeBridge is properly installed:

```javascript
import { NativeBridge } from '@nativebridge/core';

// Test the bridge connection
NativeBridge.isReady().then(ready => {
  if (ready) {
    console.log('✅ NativeBridge is installed and ready!');
  } else {
    console.error('❌ NativeBridge installation failed');
  }
});
```

## Troubleshooting

<Accordion icon="wrench" title="Common Installation Issues">
  ### iOS Build Failures

  ```bash
  # Clean and rebuild
  cd ios
  pod deintegrate
  pod install
  cd ..
  npx react-native clean
  ```

  ### Android Build Failures

  ```bash
  # Clean gradle cache
  cd android
  ./gradlew clean
  ./gradlew --stop
  cd ..
  ```

  ### Module Not Found

  ```bash
  # Clear node modules and reinstall
  rm -rf node_modules
  npm install
  # or
  yarn install
  ```
</Accordion>
